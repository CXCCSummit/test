Introduction

 

Cisco Contact Center Enterprise customers can leverage Google Contact Center Artificial Intelligence (CCAI) features like Agent Answers, in which an agent is provided with relevant suggestions and recommendations in real time for the agent to consider. The suggestions and recommendations are based on the ongoing conversation between the caller and the agent.

 

In addition, Call Transcripts services assist the agents using Artificial Intelligence (AI) and Natural Language Understanding (NLU).

These services are available for the agents in the Cisco Finesse desktop gadgets. The Call Transcript gadget dynamically converts the ongoing conversation to text and presents the text to an agent for real-time viewing and reference

 

This lab exercises outlines the steps partners or customers need to follow to implement and troubleshoot Agent Answers and Transcripts

 

Prerequisites

 

This is an intermediate session intended for engineers with prior Contact Center Enterprise (CCE) and CVP experience.

Lab 2 Create Knowledge Base and Conversation profile

 

In this lab you will be using your Google project and virtual agent in order to create the knowledge database

 

A knowledge base represents a collection of knowledge documents that you provide to Dialogflow. Your knowledge documents are provided to the Finesse agents and contain information that may be useful during the agent conversation with customers. This helps to minimize agents training and improves agents’ response to customer calls.

 

A conversation profile configures a set of parameters that control the suggestions made to an agent. These parameters control the suggestions that are surfaced during runtime. You must create a conversation profile in order to create a conversation with an end-user.

 

Note: You must have provisioned your google project and have created a virtual agent,
